# Reflection

Working on this project helped me understand how the K-Means Clustering algorithm works internally instead of relying on built-in machine learning libraries. I learned how data points are grouped based on the nearest centroid and how the centroids are updated after each iteration until the clusters become stable.

The biggest challenge was understanding how to calculate the distance between points and update the centroid values correctly. At first, the algorithm seemed difficult, but after learning the step-by-step process, it became much easier to understand.

I spent about 30 minutes understanding the problem, 1 hour designing the algorithm, 2 hours implementing the code, 45 minutes testing the output, and 30 minutes writing the documentation.

Overall, this project improved my understanding of clustering, iterative algorithms, and basic machine learning concepts. It also taught me the importance of implementing algorithms from scratch to understand how they work.
